# DuoLínguas Private School

**This is a fictional institution.**

DuoLínguas Private School is a creative and personal project — not a real school, not a registered company, not an accredited institution of any kind.

It was built as a personalized Portuguese immersion experience for one specific person, for one specific trip, in one very limited amount of time.

There are no other students. There is no waiting list. There is no curriculum.

There is only one program: **Deep Immersion BR — Accelerated Speaking & Survival Protocol**, also known, for strictly academic and absolutely respectable reasons, as the **A.S.S. Protocol**.

---

This repository contains the scheduling interface for the program — a single-page web app that displays real availability, live clocks for São Paulo and Athens, a countdown to arrival in Brazil, and direct contact options.
